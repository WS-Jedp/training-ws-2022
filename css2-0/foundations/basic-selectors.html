    With CSS we have three main ways to select any HTML element in a document, through the class, the id or the general selector of the tag.
    We need to have in mind that the class and tag selectors will affect all the elements in our HTML document, while the ID selector will affect only the element with the ID.

    To select elements that are mark with a class, we use the dot notation (.class-name)

    .header { color: white; }

    To select elements with their tag we just need to write the tag of the elements that we want to affect.

    p, h1, h3 { font-weight: bold; }

    To select an element through this ID we just need to use the # symbol followed by the name of the ID.
    #footer { widt: 100%; }
