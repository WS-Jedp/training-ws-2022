    The pseudoclasses are special selectors that will help us to add more dynamic styles to our elements, each type of pseudoelement has their own purpose.

    :hover -> The hover pseudoclass will help us to add styles when the user hovers some element with the mouse

    :focus -> The focus pseudoclass will help us to add styles when the user press some element and it is focused on it.

    :nth-child() -> The nth-child() pseudoclass will help us to add styles to the elements that fullfill the numeric rule that we put inside of the parenthesis, such as even, odd, or a numerical notation, like 2n+1

    :first -> The first pseudoclass will help us to add styles to all the first child from the element

    :target -> The target pseudoclass will help us to add styles when the user activates the target of an element in the current HTML document. The target is when in our URL it is the hash with the name of the target.
