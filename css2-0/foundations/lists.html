    We can interact and give special styles to the lists of our HTML document, we can specify the type of style that will have or even the position of the decoration.

    .list { list-style-type: circle | square | upper-roman | lower-alpha }

    We can also give an image as decoration

    .list { list-style-image: url(../../assets/images/worldskills.png); }

    Lastly we can define the position of the decoration, if it will be inside of the table element or outside.

    .list { list-style-position: inside | outside; }

    We can use the shorthand of the list-style property to pass all these attribute in one line.

    .list { list-style: square inside url(...); }
