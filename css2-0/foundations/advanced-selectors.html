    There is multiples advanced selectors that will help us to pick one or multiple specific elements of our HTML document.
    Some of them are:

    To select all the children of an element (>)

    .header > nav { ... }

    To select the first next sibling of an element (+)

    .header + .main { ... }

    To select every element that is preceed of the first element
    label ~ input { ... }
