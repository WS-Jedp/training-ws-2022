    CSS3 give us some more complex ways to select a HTML element that can be helpful at the moment of give creative styles to our website

    One of the most important selectors is searching for all the elements that has a string as the value of an attribute

    .link[href^="www.base-url.com/"] { color: red; } .title[title^="new chapter"] { color: purple; }

    Another cool selector can be the :first-letter or the :first-line that their own names self explainig about what they do.

    .text:first-letter{ font-weight: bold; color: purple; } .paragraph:first-letter{ font-weight: ligther; color: gray; }

    We can also interact with the selection generated by an user in our website through the pseudoelement ::selection.
    .long-text::selection { background-color: rgba(120, 120, 120, .21); color: white; }
    Another really useful pseduclass is the :empty one, that will add the styles to all the elements that are empty.
    .texts:empty { visibility: 0; transform: scaleX(.3) }
