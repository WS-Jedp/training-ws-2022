    PUG is a preprocessor of HTML that will help us to write a cleaner HTML, this preprocessor use JavaScript to compiole the .pug files into HTML.

    PUG help us giving us the chance of write HTML and JS in the same file, making it really dynamic at the moment of create new websites faster.

    Syntax: To write with PUG syntax we just write the name of the tag without the symbols, and to add attributes we write them into parenthesis with his value.
    The special thing about PUG is that it works with indentation, like Python, so all the elements that are one indentation more it will be considered a child of the upper element.

    p.text Hello world a.link(href='www.google.com') Go to google --- Result: <p class="text"> Hello world </p> <a class="link" href="www.google.com"> Go to google </a>
