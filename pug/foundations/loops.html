    With PUG we can make multiple loops through our JS data, writing once our HTML code but render it the times that are needed according to our data.

    each loop: One of the loops that offer us PUG is the each loop, that can iterate arrays and objects.

    - var data = ["Juanes", "Oscar"] ul.list each name, index in data a(href=`www.worldskills.org/people/?name=${name}`) li.element #{name} --- Result: <ul class="list"> <a href="www.worldskills.org/people/?name=Juanes"> <li class="element"> Juanes </li> </a> <a href="www.worldskills.org/people/?name=Oscar"> <li class="element"> Oscar </li> </a> </ul>
