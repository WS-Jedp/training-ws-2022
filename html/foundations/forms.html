    With HTML we can create forms that will help us to receive the input or data from a user, with the form we can interact with all the input that the user gives to our website or application.
    So, for start asking for information or data to the user we need to use the input tags, and for doing that we use the tag <form>, inside of this form we can use all the input data.

    There is multiple input tags that will help us to ask and recollect data from the user.

    Fieldset: There is the tag <fieldset>, that will help us to group multiple inputs or fields that are corelate to a same context data.

    <fieldset> <input type="text" name="username"> <input type="email" name="email"> </fieldset>

    Legend: This tag of <legend> will help us to show to the user some text that explains the context of the fieldset.

    <fieldset> <legend> User data </legend> <input type="text" name="username"> <input type="email" name="email"> </fieldset>

    Label: This tag of <label> will help us to create a direct context of an input, so we can use the label tag to realte some text or some information directly with an input of our form. For doing that we use the attribute for, which should have the value of the id from the input that is directly relate.

    <fieldset> <legend> User data </legend> <label for="username"> Username </label> <input type="text" name="username" id="username"> </fieldset>

    Textarea: This tag of <textarea> will help us to recollect text input data from our user, with the textarea we can specify how we are going to render this input, so we can specify the number of rows and columns that should have the input.
    The majority of input tags offer us the attribute placeholder that will help us to show to the user an example of the text or the answer that we expect to save.

    <fieldset> <legend> User data </legend> <label for="description"> Description </label> <textarea name="description" id="description" placeholder="Tell us a little bit about yourself" rows="12" col="12"> </fieldset>

    Select: This tag of <select> will help us show to the user multiple options that he could pick, the select is render as a bar of options where the user only can pick one option.
    The select tag works with the <option> tag that is the one that indicates the value and the text that the user can pick from the options input.

    The option tag can receive multiple attributes, such as the value that it will represent and also the attribute selected, that will select by default the options that has this attribute in a true value.

    <fieldset> <legend> User data </legend> <label for="gender"> Gender </label> <select name="gender" id="gender"> <option value="Male"> Male </option> <option value="Female"> Female </option> </select> </fieldset>

    Input: The input tag will help us to recollect data, the special thing about the input tag is that will help us to receive a certain type of value, such a string, a number, a boolean, etc.
    We can do that through the attribute type, that can be text, number, date, range, etc.

    <fieldset> <legend> User data </legend> <label for="birthdate"> Birthdate </label> <input type="date" name="birthdate" id="birthdate"> </fieldset>

    According to the type of the input, we can use another attributes to make quick validations with the data that the user can input to us.
    Some of these attributes are max, min, maxlength, minlength, disabled, readonly, step.
