    The anchor tag (<a>) it is an inline tag that help us to create hyper links in our website to be able to navigate through him or to redirect the user into another website.

    The anchor tag receive one fundamental attribute, href, that has the value of the URL that it will redirect the user when click it.
    Another important attribute it is the target, that indicates to the browser how it will be redirected the user, if in the same window or in a new one with the value _blank.

    <a href="www.google.com" target="_blank">Go to google</a>
