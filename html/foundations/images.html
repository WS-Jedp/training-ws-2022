    With HTML we can render images with the img tag, that receive one obligatory attribute that it is src, is the one that it will indicate the URL of the image that the document need to render.
    But there is also more important attributes such as alt, that it is a text that describes the image, this will help us to be a more accessible website and increase our visibility in the internet.

    We can specify the width of an image with the width attribute or we can use classes with CSS.

    There is a special attribute that help us to indicate multiple images and the browser will decide which one is better to render according to the weight and resolution of the device.
    This attribute is the srcset one.

    <img src='assets/images/google-logo.png' alt="Logo from google" srcset="google-logo-sm.png 320w, google-logo-md.png 480w, google-logo-lg.png 800w" sizes="(max-width: 320px) 280px, (max-width: 480px) 440px, 800px" >
