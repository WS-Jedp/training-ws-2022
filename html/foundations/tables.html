    The tables are one of the basic elements of HTML that help us to organize data or information to render it in a more organized way. The tables are compose by multiple elements, the header, the content that is is based in the rows and columns, and the footer.

    So, for create a table we use the tag <table>, inside of this we can add the another components, such the header, the content and the footer.

    All the tables works with rows and columns, we can create a column with the tag <td> and we can create a row with the <tr>.
    We need to identify the difference from the rows of the header and the rows from the content, HTML give us the tag to create the header row to indicates the titles of our table.

    <table class="table"> <thead> <tr> <th> # </th> <th> name </th> </tr> </thead> <tbody> <tr> <td> 1 </td> <td> Juanes </td> </tr> <tr> <td> 2 </td> <td> Valery </td> </tr> <tr> <td> 2 </td> <td> Valery </td> </tr> </tbody> <tfooter> <tr> <th colspan="2"> Footer of the table </th> </tr> </tfooter> </table>

    This code should render the next table.

    # Name
    1 Juanes
    2 Valery
    Footer of the table
