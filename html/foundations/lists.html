    We can create two type of lists with HTML, the ordered lists, that will help us to render ordered lists numerical and in a alphabetic way, and there is a unordered list, that will help us to render multiple elements without any specific order.

    The tag for the ordered list it is <ol> while the tag for create unordered lists it is <ul>.

    The ordered list can receive any of the three next attributes:

    • Type: Indicates how it will be ordered the list (1, A, a, i, I)
    • Start: Indicates since which element it will start the list
    • Reversed: Indicates the order of list

    Both lists, use the <li> tag to render the element.
    Use one or another type of list will help to increase the accesibility of the website which will help us to have a better SEO.

    Ordered List

    <ol type="a"> <li> a. First element </li> <li> b. Second element </li> </ol>

    Unordered List

    <ul> <li> - First element </li> <li> - Second element </li> </ul>
