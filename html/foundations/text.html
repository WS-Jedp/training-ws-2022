    So a HTML offer us multiple tags to render text content, each one with his own purpose:

    Text Tags:

    • <p>: It help us to render a paragraph, usually this are short to large texts that displays information about a topic, this tag is a block tag

    • <h1...h6>: It help us to render a title according to his hierarchy, the lowest the number the higher the importance of the title in our website. All these tags are block tags

    • <strong>: It help us to use a bold weight into a text, this tag is know as an inline tag.

    • <em>: It help us to make emphasis into a text giving it a strong weight, it is known as a inline tag.

    • <i>: It help us to render text in italic style, it is known as a inline tag.

    • <b>: It help us to render text in bold style, it is known as an inline tag.

    • <mark>: It help us to highlight a text, it is known as an inline tag.

    • <abbr>: It help us to indicate that a text, it is an abbrevation of a longer word, like NBA, it is known as an inline tag.

    • <address>: It is a tag that indicates that the content is contact information of the website, it is known as a block tag.

    • <code>: It is a tag that indicates that the content it is computer code, it is known as an inline tag.

    • <span>: It is a helper tag that will help us to interact with any piece of content wihtout create or add special styles or semantic meanings, it is known as an inline tag.
