    The script tag it will help us to create or use JavaScript in our website, we can write JavaScript in our HTML document or we can import a JS file into our document.

    The best practice is to import an external document of JS into our website, for doing that we need to know the attribute src, that it will contain the value of the URL where is our JS file.
    Also we can specify the type of code that is being include, that by default is JavaScript

    Internal Script

    <script> const msg = "Hello world" alert(msg) </script>

    External Script

    <script src="assets/scripts/main.js"></script>
